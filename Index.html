It offers a practical solution tailored to help students budget effectively and build lasting financial habits. With a Java back end integrated into a Figma-designed front end, the app provides a fully functional and cohesive experience, allowing users to navigate their finances smoothly and efficiently.

A comprehensive analysis of Pittsburgh neighborhoods to identify the best area to live in based on various data-driven criteria. Using Python, Jupyter, Pandas, and GitHub, the project focused on analyzing neighborhood data and constructing an engaging narrative within a Jupyter notebook.

MacGuffin Hunter is a fast-paced, tile-based game coded in MIPS assembly language that challenges players to navigate a grid-like arena, collecting the highly coveted MacGuffins while evading enemies. Combining strategy and quick reflexes, the game provides an engaging gameplay experience with simple yet compelling mechanics.
